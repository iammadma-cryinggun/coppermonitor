## 数据列说明

- **datetime**: 日期
- **open**: 开盘价
- **high**: 最高价
- **low**: 最低价
- **close**: 收盘价
- **volume**: 成交量
- **hold**: 持仓量
- **settlement**: 结算价

## 使用方法

### 下载数据

```bash
python download_daily_data.py
```

### 回测使用

```python
import pandas as pd

# 读取日线数据
df = pd.read_csv('daily_backtest/NI_沪镍_日线.csv')
df['datetime'] = pd.to_datetime(df['datetime'])

# 现在可以进行回测
```

## 数据来源

- 数据源: akshare (新浪期货)
- 接口: `ak.futures_main_sina(symbol)`
- 更新频率: 每个交易日收盘后
